 - **字体名称**: `arial-unicode-ms-regular` 和 `arial-unicode-ms-bold`

 ## 使用方法

 1. **下载字体文件**: 从本仓库中下载所需的 `arial-unicode-ms-regular` 和 `arial-unicode-ms-bold` 字体文件。
 2. **部署到项目中**: 将下载的字体文件放置到你的项目目录中，确保路径正确。
